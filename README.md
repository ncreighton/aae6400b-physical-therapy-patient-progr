A WordPress plugin for physical therapy clinics, rehabilitation centers, and telerehabilitation platforms. Provides therapists with a patient progress dashboard tracking functional outcome measures over time, delivers personalized home exercise programs via secure patient portal with video instruction links, sends exercise adherence reminder emails and SMS messages, collects pain and function self-report scores before appointments, and generates outcome summary reports for insurance authorization and physician communication.

== Description ==

A WordPress plugin for physical therapy clinics, rehabilitation centers, and telerehabilitation platforms. Provides therapists with a patient progress dashboard tracking functional outcome measures over time, delivers personalized home exercise programs via secure patient portal with video instruction links, sends exercise adherence reminder emails and SMS messages, collects pain and function self-report scores before appointments, and generates outcome summary reports for insurance authorization and physician communication.

**Features:**

* Core functionality

== Installation ==

1. Upload the plugin folder to `/wp-content/plugins/`
2. Activate via WordPress admin > Plugins
3. Configure via Settings > Physical Therapy Patient Progress and Home Exercise Program WordPress Plugin
